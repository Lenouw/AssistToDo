## AssistToDo 1.0.0

Première version publique. Capture vocale éclair de tâches, transcription locale, rangement intelligent.

### Fonctionnalités
- 🎙️ **Capture push-to-talk** : maintiens le raccourci, parle, relâche. Appui bref = ouvre la liste.
- 🧠 **Transcription 100% locale** (WhisperKit), hors-ligne, en français.
- 🗂️ **Rangement intelligent** (OpenRouter) : liste interne, Rappels Apple, Calendrier Apple, liste de courses dans Notes.
- 🏝️ **Îlot type Dynamic Island** ancré à l'encoche : ondes audio, états de traitement.
- 📋 **Panneau « second cerveau »** : journal de tes pensées vocales + agenda du jour iCloud.
- 📅 **Calendrier** : catégories perso/commun/pro/studio, fermeture studio en créneau timé, alarmes auto.
- 🔔 **Notifications** interactives (report, à demain, fait).
- 🔄 **Rollover quotidien** (Europe/Paris).
- ⬆️ **Vérification de mises à jour** intégrée.

### Installation
App non signée par Apple (compte développeur gratuit) : au 1er lancement, **clic-droit › Ouvrir** dans Applications. Le modèle de transcription se télécharge au premier run.

### Prérequis
- macOS 15 (Sequoia), Mac Apple Silicon.
- Une clé API OpenRouter (la tienne) à coller dans les Réglages.
